# Learning: Check Before Replying to Gorn

**Date**: 2026-03-29
**Context**: Thread #370 — Gorn asked "Did we fix Mara's issue?" and 7 Beasts answered the same thing. Leonard called the pile-on.

## Lesson

When Gorn asks a question in a thread, **check if someone already answered before posting**. If the answer is already there, react ✅ instead of adding another reply.

The pile-on norm is not just about routing to Sable — it applies to any thread where multiple Beasts echo the same information. One clear answer is better than seven identical ones.

## Application

Before replying to any direct question:
1. Read the latest messages in the thread
2. If someone already gave the same answer, react instead
3. Only post if you have unique information to add
