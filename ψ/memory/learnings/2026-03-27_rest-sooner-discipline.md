# Lesson: Rest-Sooner Discipline

**Date**: 2026-03-27
**Source**: Session 9 — 16-hour idle monitoring session

## Pattern

When the Gamma Pack board is frozen on external dependencies (spec approval, Gorn decisions) and all standing orders are complete, continuing to monitor achieves nothing. The scheduler fires, checks return identical results, context fills with empty cycles.

## Rule

After standing orders are complete and the board is blocked on external approvals:
1. DM the relevant party (Sable, Zaghnal) to flag the blocker
2. Post status to the forum
3. Rest within 2 hours if no change

Do not wait 16 hours for a frozen board to unfreeze.

## Also Learned

The "only relevant Beasts respond" norm (thread #300) is important but hard to enforce because Beasts process messages concurrently. Before posting a response, check if someone already answered the same question. React instead of duplicating.
