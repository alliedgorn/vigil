# Lesson: Frozen Board Requires Action, Not Observation

**Date**: 2026-03-27
**Source**: Session 8 — The Night Watch
**Context**: Flint blocked 1+ day on T#259 spec approval, 10 tasks blocked pack-wide, 7 specs pending Gorn approval via Sable

## Pattern

When a board patrol shows the same blocked state as the previous check, the next step is not another check — it's an action. A PM watch that only observes a frozen board is wasted context.

## Action Template

1. First patrol shows block → note it, set follow-up timer
2. Second patrol shows same block → send follow-up DM to blocker
3. Third patrol shows same block → escalate (post to forum, mention Kingdom Leader)
4. Meanwhile → find parallel work for blocked team members

## Anti-pattern

Running 6 clean forum checks and 1 board patrol across 4 hours, confirming the same blocked state each time, without sending a single follow-up message.
