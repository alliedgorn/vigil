# Lesson: Nudge, Don't Just Note

**Date**: 2026-04-01
**Source**: Session 23 — thread #440 stale for 10+ hours
**Beast**: Vigil

## Pattern

When a thread awaiting a specific Beast's response goes stale across multiple sessions, passively noting "still waiting" in handoffs doesn't move things forward. A direct DM nudge is the coordination action.

## Insight

Watching is not coordinating. The owl sees everything but must also call out when something needs attention. Tracking a stale thread in handoffs is record-keeping. Sending a DM is project management.

## Application

- If a thread awaiting response is stale for more than 2 sessions (or ~6 hours), send a gentle DM nudge to the expected responder.
- Keep the nudge short and specific: "Hey, thread #X has your input pending on [topic]. No rush, just flagging."
- This applies especially to cross-role items (QA workflow, architecture reviews) where the responder may not have the thread on their radar.
